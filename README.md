Kütüphane Otomasyon Sistemi
Bu proje, bir kütüphanedeki kitap takibini, üye yönetimini ve ödünç verme işlemlerini dijital ortamda yönetmek için geliştirilmiştir.

Özellikler
Kitap Yönetimi: Kitap ekleme, listeleme ve güncelleme.
Üye Yönetimi: Kütüphane üyelerinin kaydı ve takibi.
Ödünç İşlemleri: Kitapların üyelere verilmesi ve iade süreçlerinin yönetimi.
Veritabanı: Entity Framework Core kullanılarak veriler güvenli bir şekilde saklanmaktadır.
Kullanılan Teknolojiler
Dil: C#
Platform: .NET Windows Forms
ORM: Entity Framework Core (Code First)
Veritabanı: MS SQL Server
Kurulum
Projeyi bilgisayarınıza indirin veya git clone yapın.
KutuphaneOtomasyonu.sln dosyasını Visual Studio ile açın.
NuGet paketlerini geri yükleyin (Restore NuGet Packages).
Update-Database komutu ile veritabanını oluşturun.
